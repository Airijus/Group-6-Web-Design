WHAT ARE STATISTICS?

Statistics is the study of collecting, organising and analysing data. We use statistics to understand information and make decisions based on numbers. Statistics often requires step-by-step calculations using addition, subtraction, multiplying and dividing.

For example, when finding averages we divide, when scaling data we multiply, and when solving problems we follow clear steps to get the correct answer.

Common Calculations in Statistics

Mean = Total ÷ Number of values

Probability = Favourable outcomes ÷ Total outcomes

Ratio = Multiply or divide to make equal parts

Problem Step Answer
Find mean of 4, 6, 10 (4 + 6 + 10) ÷ 3 20 ÷ 3 = 6.67
Probability of red from 5 red, 5 blue 5 ÷ 10 1/2
Share 20 into 4 parts 20 ÷ 4 5 each
Scale data by 3 5 × 3 15

In statistics we must follow each step carefully. Multiplying and dividing correctly helps us find accurate results when working with data, averages and probability.

In A Level statistics we use data to make predictions, test ideas, and understand probability.

Explanation

  • Data can be discrete or continuous
  • We calculate mean, variance and probability
  • We use distributions to model data
  • We use hypothesis testing to make decisions

Formula

Mean = Σx / n

Variance = Σ(x − mean)² / n

Standard deviation = √variance

PROBABILITY DISTRIBUTION

A probability distribution shows all the possible values of a random variable and the probability of each value occurring.

A random variable is a variable whose value depends on chance.

Explanation

In a probability distribution:

  • Each outcome has a probability
  • All probabilities must be between 0 and 1
  • The total probability must equal 1

Probability distributions are used to model real life situations such as experiments, games, and measurements.

Formula

Mean: E(X) = Σ xP(x)

Variance: Var(X) = E(X²) − [E(X)]²

Worked Examples

Example 1

X = 1,2,3 P = 0.2,0.5,0.3
E(X) = 1×0.2 + 2×0.5 + 3×0.3 = 0.2 + 1 + 0.9 = 2.1

Example 2

E(X²) = 1²×0.2 + 2²×0.5 + 3²×0.3 = 0.2 + 2 + 2.7 = 4.9

Example 3

Var(X) = 4.9 − (2.1)² = 4.9 − 4.41 = 0.49

Example 4

Check distribution 0.3 + 0.4 + 0.3 = 1 ✔

Example 5

X = 0,1 P = 0.6,0.4 E(X) = 0×0.6 + 1×0.4 = 0.4

HYPOTHESIS TESTING

Hypothesis testing is a statistical method used to make decisions using data. We start with a claim called the null hypothesis and test it using probability calculations. We follow clear steps and use dividing, multiplying and comparing values to decide if the result is significant.

Hypothesis testing is used in mathematics, science and economics to check if results happen by chance or if there is real evidence.

Main Rules

H₀ = Null hypothesis (no change)

H₁ = Alternative hypothesis (there is a change)

Probability = favourable ÷ total

Compare probability with significance level (0.05 or 5%)

Step Action Example
1 Write hypotheses H₀: coin is fair
2 Find probability P = 1 ÷ 2
3 Multiply for outcomes (1/2) × (1/2) × (1/2)
4 Calculate result 1/8 = 0.125
5 Compare with 0.05 0.125 > 0.05 → keep H₀

Hypothesis testing is a statistical method used to make decisions based on data.

Explanation

  • H0 = null hypothesis
  • H1 = alternative hypothesis
  • Use probability to test H0
  • If probability is small → reject H0
  • If probability is large → accept H0

Formula

P(X = r) = nCr × p^r × (1 − p)^(n−r)

We start with a null hypothesis and an alternative hypothesis, then use probability to decide which one is correct.

In hypothesis testing we follow each step carefully. We multiply probabilities, divide totals, and compare results with the significance level to decide if the null hypothesis should be rejected or accepted.

Worked Examples

Example 1

n = 5 p = 0.5 r = 3 P = 5C3 × 0.5³ × 0.5² = 10 × 0.125 × 0.25 = 0.3125

Example 2

H0 : p = 0.5 Probability = 0.04 0.04 < 0.05 Reject H0

Example 3

Probability = 0.2 0.2 > 0.05 Accept H0

Example 4

n = 4 p = 0.5 r = 2 P = 6 × 0.25 × 0.25 = 0.375

Example 5

If probability < 0.05 Reject H0
