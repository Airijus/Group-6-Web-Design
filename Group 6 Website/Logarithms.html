Logarithms: The Basics

In this lesson, we will go over the basic usage cases of logarithms in mathematics.
Logarithms are the direct opposite to an exponent/indices (as taught HERE)

Logarithms ask the question: "How many times do i need to multiply a number by itself to get to another number?", and provide exact values to achieve this goal.

How do Logarithms work?

Logarithms Work using the following logic:

Logb(x) = y

where b = Base
x = Result
y = Exponent
