Logarithms: The Basics

In this lesson, we will go over the basic usage cases of logarithms in mathematics.
Logarithms are the direct opposite to an exponent/indices (as taught HERE)

Logarithms ask the question: "How many times do i need to multiply a number by itself to get to another number?", and provide exact values to achieve this goal.

How do Logarithms work?

Logarithms Work using the following logic:

Logb(x) = y

where b = Base
x = Result
y = Exponent

The equation for a logarithm can also be written as: b^y = x


Logarithms allow for precise values to be calculated when figuring out which exponent you require for specific equations

for example:

Find the value of x when 14^x = 23458967

To perform this calculation, we can identify b to be 14, and x to be 23458967
we then use these values in our logarithm equation, which gets us Log14(23458967) = y

using our calculators, we can therefore figure out that y = 6.43 to 3 significant figures.


Why is this useful?

The use of logarithms is very useful when trying to work backwards through a mathematical problem. if you are presented with a large equation containing several parts, it may be overwhelming to try and answer. using logarithms, we are able to break down the essential aspects of an equation into more bite-sized, manageable pieces.

The Laws of Logarithms

Logarithms, like their counterpart indices, have a selection of laws to follow when trying to use them. these Laws are as follows:

Multiplication Law:

Logb(x) + Logb(y) = Logb(xy)

Division Law:

Logb(x) - Logb(y) = Logb(x/y)

Power Law:

Logb(x^y) = yLogb(x)


These laws make certain logarithmic calculations significantly easier to calculate. it is common to find equations with several logarithms of the same base, which allows for the use of one of these three laws to be implemented.
NOTE: the Logs need to have THE SAME BASE for these laws to work.
